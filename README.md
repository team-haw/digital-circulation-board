## 背景
足が悪く家から出れない方や、一人暮らしの学生など町内会に参加出来ていない方を助けられるのではないかという思いから作成しました。
### コーディングについて
今回のハッカソンでは、ログイン機能やバックエンドに挑戦しました。特に、ログイン情報やプロフィール変更を反映させる所や、投稿をタイムラインに表示させる所に苦労しました。
